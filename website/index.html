Quick Installation

1

Clone the Repository

git clone https://github.com/scotthooker/OpenRouter-ImageGen-MCP.git cd OpenRouter-ImageGen-MCP
2

Install & Build

npm install npm run build
3

Get OpenRouter API Key

Sign up at OpenRouter and generate an API key from your dashboard

4

Configure Claude Code

Add to your Claude Code MCP configuration:

{ "mcpServers": { "openrouter-imagegen": { "command": "node", "args": ["/path/to/OpenRouter-ImageGen-MCP/dist/index.js"], "env": { "OPENROUTER_API_KEY": "your-api-key-here" } } } }

MacOS/Linux: ~/.config/claude-code/mcp_settings.json
Windows: %APPDATA%\Claude Code\mcp_settings.json
